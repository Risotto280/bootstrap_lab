Краткое описание

CSS (Cascading Style Sheets) - формальный язык описания внешнего вида документа.

Он используется для задания цветов, шрифтов и расположения отдельных блоков на веб-страницах.

Пример исходного кода

body {
    background-color: #f8f9fa;
    color: #212529;
}
